چنتا از اپیزود های اولش رو دیدم که در مورد خود زبان C بود و جلوتر که چطوری رفت API ویندوز رو خوند

- [React Won by Default – And It's Killing Frontend Innovation](https://www.lorenstew.art/blog/react-won-by-default/)

نوشتش که بوی AI Generated بودن می‌ده ولی چنتا نکته داره که می‌خوام بنویسم.
اشاره کرده بود به چالش استخدام دولوپر غیر ری‌اکتی. مگه ری‌اکت تخصصه؟ ری‌اکت دولوپر اصلا معنی نمیده. تو وب دولوپری دیگه.
اگه با ری‌اکت تجربه داری مگه چقدر سخته رفتن Svelte یاد گرفتن یا هر چی دیگه.
واقعا عمقی ندارن این فریمورک‌ها و تعجب می‌کنم خیلی وقتا که چرا این توهم رو داریم درست می‌کنیم که دارن.
